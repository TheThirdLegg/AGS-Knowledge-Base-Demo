Check if the power cord pig tail is unplugged from the power supply

Icon

It's normal for the top monitor to be black after a restart. Just unplug the monitor and plug it back in.

General

Multiple issues can cause the monitor to black out.

  • Check the Video card and CPU fan, are they spinning.
    • Video card is not spinning but CPU fan is, video card is the issue.
    • CPU fan is not spinning and all light on the cabinet are running. Reboot the game, if it runs for a little while and video works but the screen blacks out and the fan stops after a few minutes, the thermal sensor board on the side of the cpu heat sink could be acting up. Unplug the thermal sensor board from the IO board and see if it will stay running. 
  • Did the monitor, after power up, ever say "No Video Signal" 
    • No - if you have check everything you can think of. Look at the back of the monitor. Is the little LED lit and glowing RED. If it is, you have all the symptoms of a bad AD board. It’s the one the Video cable goes into. 
    • Yes - monitor is probably fine. You need to keep troubleshooting.  Did the motherboard make the deet-dee-deet-da-deet sound at boot?
      • No- There's your issue. You need to fix the deet-dee-deet-da-deet.   When the motherboard doesn't register the video card or ram sticks it will not send out the video signal and will not make the deet-dee-deet-da-deet noise.  Re-seat the video card and ram sticks. 
      • Yes- hmm? Try using another device as a video source for the monitor to see it it's working.  Then start swapping parts (video cable, video card, logic box).

1001-1005 Ram Clear

A 1001 through 1005 is a critical memory error. There is not always a definitive reason as to why it happened but some of the following events can cause it:

  • Improper shut down. Always shut the game down through the operator menu and after shutting off the main power switch wait for all the lights inside the cabinet to go out before turning power back on.
  • Removing the NVRam Battery.
  • Having the data logger / debug level set to anything other than zero. This can be found in the Operator Control menu. Setting the data logger level to a number higher than zero will store more info on the SSD and cause it to fill up. Only set the data logger level to a number higher than zero when instructed to by AGS TAC support.

1020 Ram Clear

Just another number for a critical memory error. Clear Chip Must be used. See 1001-1005.

1023 Progressive Values no Longer Updating

Restart the services on the progressive controller

1024 Unable to send Config to Brick

Check to insure the game is set to talk to an external progressive controller. Reboot the game. If this doesn't work, log into the progressive controller and remove the asset from the active progressive pool then delete it from quick config. Reboot the game and it will show up in quick config, then add the asset to the progressive pools.

1052 Config Error Unable to Update Config at Brick

The game is set to communicate with the progressive controller, but it. Check the network make sure you can ping the server from the game's network cable. 

  1. If the issue is with only one game, you may need to:
    1. Remove the game from the progressive pools.
    2. Delete the asset from quickconfig.
    3. Reboot the game.
    4. Re-add the game to the progressive pools.
  2. If the issue is with all the games:
    1. Check the network and switch for power and connectivity
    2. Log into the controller and make sure services are still running.

As a last resort: If the game doesn't show up in quickcofig whi showing the 1052 error you may need to replace the SSD and System CF card.

1069 Validation Error

Good Chance your SSD has a bad Sector or something is corrupt on the hard drive. If rebooting doesn't resolve the issue, replace the SSD.

1143 UPS Error

If the UPS is up and running the first thing to do is plug the usb port from the ups into one of the blue ports on the motherboard inside the logic box. You may need to reboot the game after moving the usb. If this doesn't fix the error, you may have a bad UPS or usb cable.

1144 NVRAM Battery Voltage Low

With the game powered off, spin the 3V cylinder battery behind the ribbon cable next to the video card. If the error continues to occur, you will need to replace the battery. Removing / replacing the battery will likely cause the game to go in to a Ram Clear or cause the game to request the public key (not every jurisdiction) which is why we recommend spinning the battery like you would the batteries in a TV remote first.
